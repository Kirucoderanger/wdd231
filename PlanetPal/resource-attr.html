PlanetPal - Resource Attributions

PlanetPal is built on the contributions of various resources and libraries that have enriched our project. We extend our gratitude to the following:

Data Sources

  • NASA Exoplanet Archive - For providing comprehensive datasets on confirmed exoplanets.
  • NASA API - For access to various space-related data and images.

Image Resources

  • Pixabay - For royalty-free images used in the project.
  • Unsplash - For high-quality images that enhance the visual appeal of the platform.

For more information on the resources used in this project, please refer to the Resource Attributions page.
